# Conversation Optimizer for ChatGPT

![Chrome Extension](https://img.shields.io/badge/chrome-extension-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Improve performance in long ChatGPT conversations by optimizing off-screen messages.

This browser extension reduces rendering load in large ChatGPT threads, helping keep scrolling and interaction smooth even when conversations contain many messages, code blocks, or images.

---

## Features

* **Optimizer intensity control**

  * OFF / Weak / Medium / Strong / Auto
* **Compact in-page control panel**

  * Toggle anytime using the ⚡ button
* **Automatic mode**

  * Adjusts optimization strength based on conversation size
* **Lightweight**

  * No external communication
  * No conversation data is sent anywhere

---

## Typical use cases

* Long technical discussions
* Code-heavy conversations
* Large ChatGPT threads that become slow over time

---

## Installation

### Chrome Web Store

Install from the Chrome Web Store (link will appear here once published).

### Manual installation (development)

1. Open Chrome
2. Go to:

```
chrome://extensions
```

3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the extension folder

---

## Privacy

This extension:

* does **not** send conversation data anywhere
* does **not** use external servers
* runs entirely inside the browser
